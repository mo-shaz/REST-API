API End-points and Instructions

    > POST to https://shellcorp-api.herokuapp.com/api/register to create an account in the form: { "username": name, "email": mail, "password": password }
    > POST to https://shellcorp-api.herokuapp.com/api/login to login to the created account and receive an auth token, request form: { "username": name, "password": password }
    > Copy the given auth token in the form: BEARER /AUTH_TOKEN/ in order to access the user details.
    > GET to https://shellcorp-api.herokuapp.com/api/users with the copied token in the 'Authorization' field of the header to access the user details.
